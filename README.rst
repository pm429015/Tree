About
=====

Basic Tree algorithms for python



How to use ?
==============

Declare an object Btree or BST::

    bst = BinarySearchTree()
    
Add new values to grown the tree ::

    bst.add(new values)

Delete value ::

    bst.delete(values)
    
Search value ::

    bst.search(values)
    
print result ::
    
    bst.preorder()
    bst.inorder()
    bst.postorder()
